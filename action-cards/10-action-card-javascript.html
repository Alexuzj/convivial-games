Action Cards - Random Selection with Javascript

This example introduces some Javascript that allows you to select a random 'action card'. While the code is commented, a separate Javascript tutorial will explain how the code works in detail. For now, just understand that Javascript can dynamically change or trigger different events on a page, including picking a random card.

Challenges/experiments/ideas:

  • You can style the buttons themselves!
  • What if you could have multiple 'sets' of cards -- 'action cards', 'resource cards', 'scenario cards', etc?
